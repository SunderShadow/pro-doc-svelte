# Структура проекта 

```
src/
├── lib
│   ├── assets <-- Мультимедиа (Изображения/Аудио/Видео) 
│   ├── components <-- Компоненты страниц 
│   ├── ui <-- Компоненты Ui Kit
│   │   ├── ComponentName <-- Папка с компонентом
│   │   │   ├── ComponentName.svelte <-- Файл компонента для подключения
│   │   │   └── env.scss <-- Локальные настройки компонента (Опционально) 
│   │   └── env.scss <-- Глобальные настройки UI Kit
│   └── styles
│       ├── bootstrap.scss <-- Глобальный файл со стилями
│       └── env.scss <-- Глобальный файл настройки стилей
└── routes <-- Svelte kit роутер
    └── __parts <-- Части страниц, вне UI kit
```
